What is this?

A tiny floating toolbar for describing interface changes to LLMs.

Point at any element, write what you'd change in a popover, edit text inline, then hit copy to send all your changes to your model of choice.

How does it work?

Hover Move your cursor over any element. A pink outline shows its boundaries.
Click Selects the element and opens a popover. Type what you'd change.
Edit text Click a selected text element again to edit it directly. Changes are tracked alongside your notes.
Draw Free-hand annotation mode for sketching directly on the page. Toggle from the toolbar.
Copy all Copies every annotation, text edit, and drawing as structured Markdown — ready to paste into an LLM.
Screenshot Capture the entire scrollable page as a PNG. Cmd+Shift+S or the camera button.

Shortcuts

Esc return to default tool / close popover

Esc Esc (double-tap) toggle DOM-Tools on/off

Shift+Esc clear all changes

Shift+C copy all changes as Markdown

Shift+T activate Edit Text mode

Cmd+Shift+S / Ctrl+Shift+S full page screenshot
