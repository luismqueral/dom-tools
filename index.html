DOM-Tools

What is this?

A collection of tools to help you describe interface changes to LLM's like Claude.

You can select specific DOM elements, leave sticky annotations, preview text changes and take screenshots — all to help you have more control of the UI you're crafting.

How does it work?

Hover Move your cursor around. Elements light up so you can see their boundaries.
Click Copies the element's selector and text preview straight to your clipboard.
Shift + Click Build a numbered list of elements, like pinning sticky notes to the page. All get copied together.
Hold Alt/Option Slot mode — green feedback shows insertion points. Hover near any edge (top, bottom, left, right) or center to pick a slot, click to copy.
Annotate Place sticky notes and draw freehand on the page. Press A to toggle, N/D to switch between notes and pen, X to clear.
Full Page Screenshot Capture the entire scrollable page as a PNG. Press Cmd+Shift+S or click the page icon button.

How do I install this on my project?

Copy this script and either paste it to an agent to use in your project or manually add it to your project codebase.
