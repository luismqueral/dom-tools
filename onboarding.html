Draw on the page

Copy your changes

When you're done annotating, press Shift+C or click the copy button in the toolbar. This bundles all your selections, text edits, and drawings into structured Markdown you can paste straight into an LLM (ChatGPT, Claude, etc).

5

Useful shortcuts

A few more things to know:

Esc — return to default tool / deselect

Esc Esc — toggle DOM-Tools on/off

Shift+Esc — clear all annotations

Shift+C — copy all changes

Shift+T — enter text edit mode

Cmd+Shift+S — full-page screenshot

Try clearing all your changes with Shift+Esc to reset the page.
