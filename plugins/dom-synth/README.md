# DOM Synth

Turn the page into a musical instrument. Web Audio API synthesis, zero dependencies.

## Modes

- **Hover** — elements produce sound as you move over them
- **Sequence** — click elements to build a step sequencer pattern
- **Drone** — selected elements sustain as layered oscillators
- **Theremin** — mouse position maps to pitch and volume continuously

## Scales

Chromatic, major, minor, pentatonic, blues, dorian, japanese, and whole-tone.

## Enabling

Load after the main DOM-Tools script:

```html
<script src="dom-tools.js"></script>
<script src="plugins/dom-synth/dom-synth.js"></script>
```

Then toggle it on in DOM-Tools settings (gear icon).
