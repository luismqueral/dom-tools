# Spacing Debugger

Page-wide margin and padding overlay plugin for DOM-Tools. Shows spacing for all visible elements simultaneously — no hover required.

## Usage

When active, margins render in orange and paddings in green across the entire viewport. A small control panel lets you toggle between margin-only, padding-only, or both, and optionally show pixel labels.

## Enabling

Load after the main DOM-Tools script:

```html
<script src="dom-tools.js"></script>
<script src="plugins/spacing-debugger/spacing-debugger.js"></script>
```

Then toggle it on in DOM-Tools settings (gear icon).
