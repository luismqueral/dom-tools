# DOM X-Ray

Box-model visualization plugin for DOM-Tools. Shows content (blue), padding (green), border (yellow), and margin (orange) as colored overlays on any element.

## Usage

Hover over elements to see their box model. Click to lock a selection and inspect the computed values in a tooltip.

## Enabling

DOM X-Ray is an optional plugin. Load it after the main DOM-Tools script:

```html
<script src="dom-tools.js"></script>
<script src="plugins/dom-xray/dom-xray.js"></script>
```

Then toggle it on in DOM-Tools settings (gear icon).
